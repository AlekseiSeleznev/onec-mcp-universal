# /1c-project-init

Bootstrap-скилл для локальной подготовки проекта 1С под текущий стек `onec-mcp-universal`.

## Цели

1. Проверить структуру проекта и минимальные требования.
2. Добавить workflow-артефакты (`session-notes.md` template, docs pointers).
3. Подготовить/обновить проектный `.mcp.json` на основе `templates/mcp.json`.
4. Не трогать инфраструктурно-специфичные настройки (конкретные серверы/пароли/CT*).

## Шаги

1. Определи target-path (по аргументу или текущая директория).
2. Проверь наличие ключевых папок (`src`, `ext`, `skills`, или эквивалентов проекта).
3. Скопируй `templates/mcp.json` в `<target>/.mcp.json`, если файла нет.
4. Если `.mcp.json` уже есть — не перезаписывай, а предложи merge рекомендаций.
5. Создай (если отсутствует) `session-notes.md` с базовой структурой.
6. Выведи отчёт bootstrap: что создано, что пропущено, что требует ручной настройки.

## Ограничения

- Не выполнять деплой/восстановление баз.
- Не запускать 1cv8/DESIGNER напрямую.
- Не записывать секреты в репозиторий.

## Результат

Проект готов к работе с:
- `onec-mcp-universal` transport,
- workflow skills (`session-*`, `*-expert`),
- безопасным ручным заполнением чувствительных параметров.
