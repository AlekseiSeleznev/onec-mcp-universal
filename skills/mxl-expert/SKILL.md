# /mxl-expert

Оркестратор работы с MXL-макетами и их подключением к объектам.

## Поддерживаемые операции

- Скомпилировать MXL из JSON -> `/mxl-compile`
- Декомпилировать MXL в JSON -> `/mxl-decompile`
- Добавить макет в объект -> `/template-add`
- Удалить макет из объекта -> `/template-remove`
- Проверить макет -> `/mxl-validate`
- Проанализировать структуру -> `/mxl-info`

## Правила

1. При изменении макета сначала выполняй compile/decompile, затем проверку.
2. При добавлении/удалении макета на объекте заверши проверкой `/mxl-validate`.
3. Если у пользователя нет явной цели, сначала выполни `/mxl-info` для диагностики.
