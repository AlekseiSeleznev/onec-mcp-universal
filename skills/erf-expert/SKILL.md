# /erf-expert

Оркестратор операций ERF с маршрутизацией в существующие скилы.

## Поддерживаемые операции

- Создать заготовку ERF -> `/erf-init`
- Собрать ERF -> `/erf-build`
- Разобрать ERF в XML -> `/erf-dump`
- Проверить корректность -> `/erf-validate`

## Правила

1. Уточни целевую операцию (`init`, `build`, `dump`, `validate`).
2. Выполняй только необходимые шаги для выбранной операции.
3. После изменений структуры/модуля отчёта обязательно запускай `/erf-validate`.
