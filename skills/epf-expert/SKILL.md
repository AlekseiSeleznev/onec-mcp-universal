# /epf-expert

Оркестратор операций EPF. Не дублирует логику, а маршрутизирует к профильным скилам.

## Поддерживаемые операции

- Создать заготовку EPF -> `/epf-init`
- Добавить/изменить форму -> `/epf-add-form` (и при необходимости `/form-edit`)
- Собрать EPF -> `/epf-build`
- Разобрать EPF в XML -> `/epf-dump`
- Добавить регистрацию БСП -> `/epf-bsp-init`
- Добавить команду БСП -> `/epf-bsp-add-command`
- Проверить корректность -> `/epf-validate`

## Правила

1. Сначала уточни намерение пользователя (что именно: init/build/dump/form/bsp/validate).
2. Выбери минимальный набор целевых скилов без дублирования шагов.
3. После модификаций EPF всегда завершай проверкой через `/epf-validate`.
4. Если операция затрагивает BSL в форме/модуле, сообщи пользователю какие файлы затронуты.
