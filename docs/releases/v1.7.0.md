# onec-mcp-universal v1.7.0

Дата: 2026-04-14

## Что вошло

- `export_bsl_sources` для MCP переведён в безопасный **background-by-default** режим.
  Долгие выгрузки больше не упираются в клиентский `tools/call` timeout.
- Добавлен новый шлюзовый инструмент `get_export_status` для проверки статуса
  фоновой выгрузки и индексации.
- Исправлен regression в `mcp_server` при импорте без предварительной инъекции
  `manager/registry` из `server.py`.
- Сохранён и покрыт тестами MCP-контракт `export_bsl_sources`:
  `output_dir="/projects"` теперь корректно разворачивается в настроенный host workspace.
- Усилен тестовый контур ретеста MCP-сервера, BSL export flow и анонимизации proxy-ответов.
