# onec-mcp-universal v1.6.0

## Highlights

- Removed legacy static `mcp-lsp-zup` path from the gateway runtime contract.
- Made `bsl-lsp-bridge` reproducible on fresh machines via repo-local `tools/build-lsp-image.sh`.
- Switched documented and automated LSP build path to Java 21 compatible runtime.
- Fixed per-database LSP path mapping and `did_change_watched_files(language="bsl")` calls.
- Persisted and auto-restored `bsl_search` index after export and gateway restarts.
- Improved dashboard diagnostics for optional services and EPF/LSP state.
