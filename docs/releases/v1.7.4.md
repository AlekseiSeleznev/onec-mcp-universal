# onec-mcp-universal v1.7.4

## Что вошло

- Формальный patch-релиз поверх `v1.7.3`, чтобы релизный тег снова совпадал с актуальным `main`.
- Закрыта гонка при удалении базы из дашборда:
  - `remove` теперь дожидается фактической остановки toolkit/LSP контейнеров базы до возврата успешного ответа;
  - мгновенное повторное подключение той же базы больше не должно гоняться с фоновым cleanup.
- Добавлен регрессионный тест на синхронный runtime cleanup при `remove`.
