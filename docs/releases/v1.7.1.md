# onec-mcp-universal v1.7.1

## Что вошло

- Dashboard-применение настроек BSL workspace теперь обновляет runtime-конфиг шлюза без ручного перезапуска.

- Добавлены тесты на обновлённую диагностику и runtime-применение gateway-настроек.
