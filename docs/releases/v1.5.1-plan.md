# onec-mcp-universal v1.5.1 Plan (Tech)

Дата: 2026-04-13

## Goal

Снизить стоимость сопровождения без изменения внешнего MCP API.

## Scope

1. Декомпозиция `gateway/gateway/mcp_server.py`:
- выделить каталог tool-handlers (`tool_handlers/*.py`);
- оставить в `mcp_server.py` только registration + dispatch.

2. Декомпозиция `gateway/gateway/server.py`:
- вынести lifecycle/startup/cleanup в отдельные модули;
- вынести dashboard action handlers по группам.

3. Test/ops hardening:
- добавить targeted regression tests на новые boundaries после декомпозиции;
- зафиксировать release-checklist для CI + smoke + load-smoke.

## Non-goals

- Изменение MCP tool contracts.
- Изменение пользовательских endpoint URL.

## Exit criteria

- Поведение API неизменно (green regression).
- Full test suite green.
- Документация синхронизирована с фактической архитектурой.
